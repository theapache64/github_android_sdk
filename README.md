# github_android_sdk
An android SDK for github (beta)

### Install

```groovy
    compile 'com.theapache64.github_android_sdk:github_android_sdk:1.0.0'
```

### Init

```
    GitHubAPI.init("YOUR_GITHUB_ACCESS_TOKEN");
```

the rest is self explanatory.

### Supported operations

- Create Issue
- List Issues
- Create Comment
